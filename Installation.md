# インストール手順

## Microsoft Edge の場合

### Windows11 ( 2023年11月現在 )

`MemoPad.msi` をクリックすると `開く ( Open )` か `名前を付けて保存 ( Save as )` かの二択を迫られます.

`開く ( Open )` を選ぶと,
ダウンロードした後に `保存 ( Keep )` か `削除 ( Delete )` の二択になります.
ここで `保存 ( Keep )` を選ぶといきなりインストーラーが走り出すので,
インストーラーが提示してくるデフォルトを変える必要がなければ,
`Next>` ボタンを 3回クリックするだけです.
すると,
ユーザーアカウント制御 ( User Account Control ) が管理者権限を求めてくるので,
了承するとインストールが済んで `Close` ボタンを押すばかりになるので,
クリックすればインストールは完了です.

`名前を付けて保存 ( Save as )` を選ぶと,
ダウンロードした後に `保存 ( Keep )` か `削除 ( Delete )` の二択になります.
ここで `保存 ( Keep )` を選ぶとダウンロードは終わります.
で, ここで現れた `ファイルを開く ( Open file )` をクリックするか落とした `MemoPad.msi` をダブルクリックすると,
ようやくインストールが始まります.
インストーラーが提示してくるデフォルトを変える必要がなければ,
`Next>` ボタンを 3回クリックするだけです.
すると ユーザーアカウント制御 ( User Account Control ) が許可を求めてくるので,
了承するとインストールが済んで `Close` ボタンを押すばかりになります.
このボタンをクリックすればインストールは完了です.

### Windows10 ( 2023年11月現在 )

`MemoPad.msi` をクリックするとブロックされることがあります.

その場合, 「ブロックしたよ。」との表示にマウスをあてがうと右端にゴミ箱のアイコンと `…` が現われます.
その `…` をクリックするとさらに別のメニューが現われます.
そこには `保存 ( Keep )` という選択肢があるのでこれを選びましょう.
するとようやくすなおにダウンロードが行われます. ( 以上 Microsoft Edge の場合 )

で, その場で `ファイルを開く ( Open file )` をクリックするか落とした `MemoPad.msi` をダブルクリックすると,
「Windows によって PC が保護されました ( Windows protected your PC )」とのことで,
`実行しない ( Don't run )` というボタンをデフォルトで推してきます.
しかし, よく見ると `詳細情報 ( More info )` というリンクがあるのでこれをクリックすると,
`実行 ( Run anyway )` というボタンが現われます.
このボタンをクリックするとようやくインストールが始まります.
インストーラーが提示してくるデフォルトを変える必要がなければ,
`Next>` ボタンを 3回クリックするだけです.
すると,
ユーザーアカウント制御 ( User Account Control ) が許可を求めてくるので,
了承するとインストールが済んで `Close` ボタンを押すばかりになります.
このボタンをクリックすればインストールは完了です.
